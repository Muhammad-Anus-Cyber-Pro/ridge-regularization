# Ridge Regression with Hyperparameter Tuning

## 📌 Overview
This project demonstrates the implementation of Ridge Regression on a real-world dataset to handle overfitting and improve model performance.

## 📊 Dataset
California Housing Dataset (sklearn)

## ⚙️ Techniques Used
- Linear Regression (Baseline)
- Ridge Regression
- Hyperparameter Tuning (GridSearchCV)
- Cross Validation (cv=5)
- Feature Scaling

## 🎯 Results
- Compared Linear vs Ridge Regression
- Identified optimal alpha value
- Improved model generalization

## 🧠 Key Learnings
- Regularization helps reduce overfitting
- Proper alpha selection is crucial
- Cross-validation ensures robust performance
